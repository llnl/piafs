About PIAFS
===========

PIAFS (Photochemically Induced Acousto-optics Fluid Simulations) is a finite-difference code for solving compressible Euler and Navier-Stokes equations on Cartesian grids.

Features
--------

* Compressible Euler and Navier-Stokes equations (1D, 2D, 3D)
* High-order spatial discretization schemes
* MPI parallelization
* GPU acceleration (CUDA and HIP)
* Photochemistry module
