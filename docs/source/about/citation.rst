Citing PIAFS
============

If you use PIAFS in your research, please cite the following publication:

Primary Citation
----------------

Oudin, A., Ghosh, D., Riconda, C., Lancia, L., Kur, E., Ou, K., Perez-Ramirez, V. M., Lee, J., Edwards, M. R., & Michel, P. (2025). PIAFS: A 2D nonlinear hydrodynamics code to model gaseous optics. *Physics of Plasmas*, 32(7), 072714.

**DOI:** https://doi.org/10.1063/5.0268318

**Full URL:** https://pubs.aip.org/aip/pop/article/32/7/072714/3355887/PIAFS-A-2D-nonlinear-hydrodynamics-code-to-model

BibTeX
------

.. code-block:: bibtex

   @article{oudin2025piafs,
     title = {PIAFS: A 2D nonlinear hydrodynamics code to model gaseous optics},
     author = {Oudin, A. and Ghosh, D. and Riconda, C. and Lancia, L. and
               Kur, E. and Ou, K. and Perez-Ramirez, V. M. and Lee, J. and
               Edwards, M. R. and Michel, P.},
     journal = {Physics of Plasmas},
     volume = {32},
     number = {7},
     pages = {072714},
     year = {2025},
     month = {July},
     doi = {10.1063/5.0268318},
     url = {https://doi.org/10.1063/5.0268318}
   }

   @misc{piafs_software,

     url = {https://github.com/LLNL/piafs},
     note = {LLNL-CODE-2015997}
   }

Acknowledgment
--------------

When acknowledging PIAFS in publications, consider using text like:

   "Simulations were performed using PIAFS [cite], a finite-difference code for
   compressible Euler/Navier-Stokes equations developed at Lawrence Livermore
   National Laboratory."
